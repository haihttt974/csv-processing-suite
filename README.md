# Bộ công cụ xử lý CSV

Ứng dụng dòng lệnh hỗ trợ nhiều chức năng thao tác dữ liệu CSV bằng Python.

## Cài đặt

```bash
pip install -r requirements.txt
```

## Sử dụng

```bash
python main.py
```

Sau khi đăng nhập thành công (mật khẩu mặc định `haine`), chương trình yêu cầu nhập
đường dẫn tới file CSV và hiển thị menu chức năng.

## Danh sách chức năng

1. Xóa chuỗi trong cột hoặc toàn bộ file
2. Thống kê giá trị duy nhất của cột
3. Tìm và thay thế chuỗi
4. Chuyển hoa/thường dữ liệu
5. Xóa dòng rỗng hoặc chứa chuỗi
6. Gộp hoặc tách cột
7. Lọc dữ liệu theo điều kiện đơn giản
8. Thống kê số (tổng, trung bình, min, max)
9. Sắp xếp dữ liệu theo cột
10. Xử lý thời gian và chuẩn hóa định dạng
11. Xuất dữ liệu ra JSON hoặc TXT
12. Kiểm tra và sửa lỗi dữ liệu
13. Xem nhanh 5 dòng đầu/cuối
14. Đổi đường dẫn file đang làm việc
15. Vẽ biểu đồ line/bar/pie
16. Xóa dòng trùng lặp
17. Gộp nhiều file CSV
18. Tách file CSV thành nhiều phần
19. Tạo báo cáo tổng hợp tự động
20. Tạo cột mới dựa trên công thức
21. Chuẩn hóa dữ liệu trong cột
22. Phát hiện loại dữ liệu của cột
23. Trích xuất bảng từ Excel/PDF
24. Lọc nâng cao bằng biểu thức Python

## Yêu cầu

- Python 3.11 trở lên
- Các thư viện trong `requirements.txt`
